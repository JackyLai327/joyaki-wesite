To start the game...

After downlaod, unzip the folder:

1. Open a new terminal/command prompt.

2. Drag the game folder into the terminal/command prompt, then hit enter.

3. Type the following command into command line and hit enter, then enjoy the game:

python3 minigame-1/main.py

If your PC/laptop doesn't have python installed or updated, please check out the link: Python Install/Update Guide
